 <h3>About the Core Blog Module</h3>
 <p class="Owner">
  <strong>Version 6.0.0</strong><br /><br />
  This version of the module requires DotNetNuke Core 07.00.04 or greater. <br /><br />
  FYI: Developers should be aware that the module uses Visual Studio 2010 only. <br /><br />
  Release Highlights:
  <ul>
   <li>Complete rewrite of the module</li>
   <li>UI completely based on templates</li>
   <li>Bloggers can have any number of blogs</li>
   <li>Blogs now tied to module instead of portal</li>
   <li>Every blog has a set of permissions that govern ghost writing and commenting</li>
   <li>Just one single module definition now</li>
   <li>You can point the module to another to use the other's blog content</li>
   <li>"Hybrid multilingual capable module":
  <ul>
   <li>Full localization: every blog post has optional translations</li>
   <li>Regular: every blog post is labeled with a locale</li>
  </ul>
</li>
   <li>Categories and tags can be translated</li>
   <li>Select an image for each blog and post</li>
   <li>Proper editor for categories</li>
   <li>BlogML import and export</li>
   <li>DNN 7 services layer to make a much more fluent UI</li>
  </ul>
